arkouda.util.expand(size, segs, vals)#

Expand an array with values placed into the indicated segments.

Parameters:

  • size (ak.pdarray) – The size of the array to be expanded

  • segs (ak.pdarray) – The indices where the values should be placed

  • vals (ak.pdarray) – The values to be placed in each segment

Returns:

The expanded array.

Return type:

pdarray

Notes

This function (with different order of arguments) is now in arkouda proper as ak.broadcast. It is retained here for backwards compatibility.

arkouda.util.unregister_by_name(name: str, dtype: str = 'infer')#

Unregisters all components of a given registered object’s name

Parameters:

  • name (str) – Name of the object to be unregistered

  • dtype (str) – Name of the type of object being unregistered. If no type is given, we will attempt to identify the type based on registered symbols Supported types are: pdarray, strings, categorical, segarray, and series
