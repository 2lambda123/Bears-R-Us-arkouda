Usage

use BinOp;

or

import BinOp;
const omLogger = new Logger(logLevel)¶
proc doBinOpvv(l, r, e, op: string, rname, pn, st) throws¶

Generic function to execute a binary operation on pdarray entries in the symbol table

Arguments

  • l – symbol table entry of the LHS operand

  • r – symbol table entry of the RHS operand

  • e – symbol table entry to store result of operation

  • op : string – string representation of binary operation to execute

  • rname : string – name of the e in the symbol table

  • pn : string – routine name of callsite function

  • st : borrowed SymTab – SymTab to act on

Returns

(MsgTuple)

Throws

UndefinedSymbolError(name)
